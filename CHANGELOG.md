# Changelog

All notable changes to this project will be documented in this file.

## v1.0.0 - 2026-03-27

### Added

- Full-stack library management workflow with FastAPI backend and React frontend
- Role-based login and dashboard structure for admin and user journeys
- Maintenance modules for memberships, catalog items, and user management
- Transaction flows for search, issuing, returns, and fines
- Reporting views for books, movies, memberships, active issues, overdue returns, and requests

### Changed

- Backend configuration aligned to PostgreSQL for local development
- Frontend application expanded to cover workbook-driven business flows and polished styling

### Notes

- This release is intended as the initial public GitHub version of the project
- Local setup instructions are documented in `README.md`
